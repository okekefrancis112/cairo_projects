# Cairo Projects

A collection of Cairo and StarkNet smart contract projects for learning and experimentation.

## Overview

A workspace for developing and testing Cairo smart contracts on StarkNet, covering various contract patterns and DeFi primitives.

## Tech Stack

- **Cairo** — StarkNet smart contract language
- **TypeScript** — Testing and deployment scripts
- **StarkNet** — L2 scaling solution

## Getting Started

```bash
git clone https://github.com/okekefrancis112/cairo_projects.git
cd cairo_projects
npm install
```
